# 마케팅 실행 스펙: 첫 30개 소개서 — 컨시어지 선물 패키지

날짜: 2026-07-20 · 상태: 대표 설계 승인(대화) → 스펙 리뷰 대기
배경 정본: 노션 기획서 §12~13(North Star 카드뷰·씨딩 4스텝·캔가 파트너 보호) · 볼트 [[BM-전략]](90일 컨시어지·MD pull·허영지표 경계)

## 목표

씨딩 스텝 1(지인)~1.5(확장 시동)를 실행해 **접수(claim)된 소개서 30개**를 모은다.
코어 루프 = 대표가 지인 가게 소개서를 **미리 만들어 선물**(링크+지류) → 지인이 **자기 것으로 접수·수정** → **링크인바이오(링크트리·리틀리·인스타 바이오)에 걸게** 한다.

- 진실 지표 = **링크인바이오 등록 수** (등록·가입 수는 허영지표 — 세지 않음)
- 선물 프레이밍(대표 확정): "직접 작성" 대신 **완성품 리빌**이 첫 만남의 매직. "이건 초안 — 네 말투로 바꿔, 수정 쉬워"가 소유감 장치. 직접 작성(위저드) 매직 검증은 확장 단계의 낯선 유저에게서 공짜로 얻는다.

## 구조 — 3주 프로그램 + 확장

```
STEP 0 (1~2일)   사전 준비 — 리허설·스크립트·지류 템플릿
STEP 1 (1주차)   파일럿 6곳 (대표 지정: 바로 컨택 가능)
   🚪 게이트     ①접수 3+/5 (1주차 말 평가 — STEP 2 진행 조건)
                ②바이오 2+/5 (2주차 말까지 롤링 평가 — 바이오는 접수보다 며칠 늦는 게 정상)
                ※캔가는 분자·분모 모두 제외(와이프 접수는 보장된 값 — 게이트 분식 방지). 플로우 리허설로만
STEP 2 (2주차)   나머지 6곳
STEP 3 (3주차~)  팔로업 + 확장 시동(지인의 지인·편집숍 리스트업) → 30개
```

## 선물 패키지 — 곳당 표준 사이클 (7단계)

1. **제작**: ⚠️ **반드시 로그아웃 상태**로 위저드 등록 + **곳별 고유 수정 비밀번호**(지인이 기억하기 쉬운 것, 예: 가게이름 축약+숫자). 로그인 제작 시 대표 계정 귀속 → 이관 불가(기술 확인됨). 홈페이지 있으면 링크 확정 단계에서 넣기(딥리드).
2. **다듬기**: 크롤 초안을 대표가 검수 — 사실 오류 제거·사진(지인에게 몇 장 요청 가능)·한 줄/자세히 후보 선택.
3. **지류**: 소개서를 수동으로 보기 좋게 인쇄(캡처 배치) + 클리어파일 + QR코드(페이지 링크). 인쇄 CSS 개발 안 함 — 수동이 v1.
4. **전달**: ⭐어차피 만날 곳 = 대면(+그 자리에서 **같이 5분 수정** — 리빌 놀라움+수정 재미+즉석 접수) / 나머지 = 카톡(링크+비번+지류는 다음 방문 때).
   **피치 스크립트(고정 3줄)**:
   - "네 가게 소개 페이지를 만들어봤어 — 지금 그대로 포트폴리오·링크트리에 쓸 수 있어." (지금 가치)
   - "이건 초안이야 — 네 말투로 바꿔줘. 비번만 넣으면 수정 진짜 쉬워." (소유감 장치)
   - "나중에 바이어·MD 몇 분께 네 페이지 보여줘도 돼?" (컨시어지 사전 동의 — 90일 플레이북 대비)
   - (구두로만) "다음 단계로 결 맞는 브랜드끼리 이어주는 걸 만들 거야. 너희가 초기 멤버." — 제품·카피엔 과약속 금지.
5. **접수 안내**: 수정해보게 함 → 가입 시 /my "연결하기"(링크+비번)로 본인 계정 귀속(claim 플로우 배포돼 있음).
6. **바이오 요청(2박자)**: 전달 시엔 가벼운 씨앗만 — "나중에 링크트리에 넣으면 딱이야." **본 요청은 접수(claim) 후 D+1~2에 카톡** + 바이오 가이드 1장과 함께 — 아직 자기 것도 아닌 페이지를 홍보해달라면 선물 프레임이 깨지고 '조건'처럼 읽힌다.
7. **기록**: 트래킹 표에 [전달일·접수·바이오·반응 메모] 기입.

## 지표·게이트

| 지표 | 파일럿(캔가 제외 5) | 전체(캔가 제외 11) |
|---|---|---|
| 접수(claim) | 3+ (1주차 말) | 9+ |
| **링크인바이오** ⭐ | 2+ (2주차 말까지 롤링) | 5+ |
| 정성 | "진짜 쓸게" 자발 발화 기록 | 동일 |

- 접수 게이트만 STEP 2 진행을 막는다. 바이오는 접수보다 자연히 며칠 늦으므로 롤링 평가 — 대리지표("걸겠다는 약속")로 대체 금지.
- 게이트 미달 시: STEP 2 진행 전 원인 수정(피치? 접수 UX? 품질?) — 지인 자산은 리필 불가라 문제 안고 확산 금지.
- ⭐(대면 가능) 수가 5 미만이면 가까운 3~5곳은 일부러 방문 일정을 잡아 대면 티어 유지.

## 가정 (명시 — 리뷰 게이트용)

- **시간**: 마케팅 주 15~20h(풀타임이지만 제품 개발 병행 — 대표 답변 40~50h의 보수 계산). 곳당 ~2~2.5h → 파일럿 6곳 = 1주 안.
- **예산**: 지류 곳당 ~5천원(컬러 양면+클리어파일+QR스티커), 12곳 ≈ **6만원**. 특별한 곳만 소폭 업그레이드.
- **개발 0**: 접수·수정·전달 플로우 전부 배포됨. 파일럿에서 실막힘 나올 때만 개발(후보: 인쇄 CSS·접수 안내 문구·OG 이미지).

## 룰·리스크

- **캔가 파트너 보호 = 상시 필터**: 이 리스트뿐 아니라 확장 대상(편집숍 포함) 선정 시마다 와이프 확인.
- 지인 반응 할인: 말 아닌 **행동**(바이오·재방문 수정)만 신호.
- 크롤 오류가 지인 앞에서 나오면 즉시 제품 피드백 루프(이번 주 크롤 대공사의 실전 QA를 겸함).

# ✅ Step-by-Step 체크리스트 (대표 실행 순서)

## STEP 0 — 사전 준비 (1~2일)

- [ ] **0-1. 접수 리허설**: 로그아웃 → 테스트 소개서 1개 위저드 등록 + 비번 설정 → 별도 계정으로 로그인 → /my "연결하기"에 링크+비번 → 귀속 확인 → 테스트 소개서 삭제(대표 직접)
- [ ] **0-2. 캔가 소유 확인**: 캔버스가든 페이지가 어느 계정에 묶여 있는지 확인 — 와이프 계정으로 옮길지/그대로 둘지 결정
- [ ] **0-3. 지류 리허설**: 캔가(또는 데모) 소개서로 인쇄 1회 → 보기 좋은 캡처·배치 확정 → 클리어파일 12개·QR스티커 구매(예산 ~6만원)
  - 폴백 체인(막히면): 집 인쇄 별로 → 동네 인쇄소 소량 출력(곳당 5천원 내) → 그래도 별로 → 파일럿은 디지털 우선, 지류는 후속 선물로
- [ ] **0-3b. 바이오 가이드 1장 제작**: 링크트리·리틀리·인스타 바이오에 링크 넣는 법 캡처 가이드 — 접수 후 D+1~2 요청 때 첨부(1~2주차 의존물이라 STEP 0로 이동)
- [ ] **0-4. 피치 스크립트 확정**: 위 3줄+구두 로드맵을 자기 말투로 다듬어 저장(카톡 보낼 기본 문구 포함)
- [ ] **0-5. 트래킹 표 만들기**: 옵시디언 마케팅 노트에 12곳 × [링크·비번·전달일·접수·바이오·메모] 표 생성
- [ ] **0-6. ⭐ 달력 매칭**: 파일럿 6곳 중 2~3주 내 자연스럽게 만날 곳 표시, 안 만날 곳은 방문/카톡 결정

## STEP 1 — 파일럿 6곳 (1주차)

곳마다 사이클 7단계 실행. 추천 순서(쉬운→덜 쉬운):
- [ ] **1-1. 피망당구클럽** (아빠 — 리스크 0, 크롤 검증 완료. 첫 실전으로 최적) — ⏱ 제작~전달 소요시간 기록
- [ ] **1-2. 캔버스가든** (와이프 — 기존 페이지 최신화 + 지류 + 바이오. 게이트 산식 제외, 쇼케이스 예시용) — ⏱ 소요시간 기록 → **2곳 실측으로 곳당 2~2.5h 가정 재보정 후 STEP 1 나머지 일정 확정**
- [ ] **1-3. 상왕제약 홍익점**
- [ ] **1-4. 상왕라이브러리** (같은 상왕 계열이면 같은 날 묶어 전달)
- [ ] **1-5. 두더지요가원**
- [ ] **1-6. 아그레아블**
- [ ] **1-7. 게이트 평가**: 접수 4+? 바이오 2+? 반응 메모 정리 → 미달 항목 원인 수정 후 STEP 2

## STEP 2 — 나머지 6곳 (2주차)

- [ ] 2-1. 콜렉트마이페이보릿 (크롤 QA 완료 — 예열됨)
- [ ] 2-2. 호락호락도서관 · 2-3. 호락호락작업 (같은 팀이면 묶음 전달)
- [ ] 2-4. 캔앤코르크(⚠️캔가 파트너 여부 와이프 확인 후) · 2-5. 로컬페이지 · 2-6. 계단뿌셔클럽
- [ ] 2-7. 전체 지표 점검(접수 10+/바이오 5+)

## STEP 3 — 팔로업 + 확장 시동 (3주차~)

- [ ] 3-1. 미접수 곳 팔로업(가볍게 1회 — "고쳐줄 거 있으면 말해")
- [ ] 3-2. 바이오 미등록 곳 리마인드(가이드는 STEP 0-3b에서 이미 제작 — 재발송만)
- [ ] 3-3. **지인의 지인 요청**: 반응 좋았던 지인에게 "주변에 이런 거 필요한 사장님 있어?" (소개 스크립트 준비)
- [ ] 3-4. **편집숍 리스트업**: 캔가 입점처·아는 편집숍 5곳 후보화 — ⚠️**5곳 각각** 캔가 파트너 여부 와이프 확인(체크 없인 컨택 금지)
- [ ] 3-5. 30개 도달 시 [[BM-전략]] 90일 컨시어지(MD 영업)로 전환 — MD 노출 동의는 이미 수집됨
